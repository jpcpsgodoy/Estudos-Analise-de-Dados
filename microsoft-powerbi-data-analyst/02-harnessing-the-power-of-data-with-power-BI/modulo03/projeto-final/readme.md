# Módulo 3: Análise e Processamento de Dados

## Preparação para o Lançamento
Nesta etapa, o foco foi a análise técnica dos dados de vendas da Adventure Works para suportar o lançamento de uma nova linha de produtos.

## O que foi feito no Projeto Final:
* **Identificação de Stakeholders:** Definição das necessidades das equipas de Vendas, Marketing e Desenvolvimento de Produtos.
* **Análise de Fontes de Dados:** Exploração do dataset `Sales.csv` para extrair detalhes sobre tendências de vendas e perfis de clientes.
* **Limpeza de Dados no Excel:** Aplicação de funções (como ARRUMAR e filtros) para tratar inconsistências e remover campos vazios.
* **Tipagem de Dados:** Padronização das colunas (Número, Texto, Moeda, Data) para garantir a integridade da análise.

## Por que foi feito:
* **Suporte à Decisão:** Para que o Marketing e as Vendas baseiem as suas estratégias em dados reais de preferências dos clientes.
* **Qualidade da Informação:** A limpeza (ETL) garante que erros nos dados brutos não levem a decisões de negócio incorretas.
* **Visão 360º:** A consideração de outras fontes (feedback e analytics) permite entender não só o "quanto" se vendeu, mas o "porquê".

## Conclusão
Este processo estabeleceu a base de confiança nos dados necessária para guiar o sucesso do novo produto no mercado.
