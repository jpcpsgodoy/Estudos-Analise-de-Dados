# Módulo 2: Preparação e ETL

## A Paleta do Artista
Assim como um artista prepara suas tintas, o analista usa o **ETL** para garantir que os dados estejam limpos e prontos para a tela do Power BI.

## Conceitos-Chave:
* **Fluxo Power BI:** Desktop (criação), Service (partilha) e Apps (consumo).
* **Processo ETL:** Extrair, Transformar e Carregar dados com foco em governança.
* **Limpeza:** Tratamento de nulos, duplicatas e erros de tipo de dado.
* **Power Query:** Automação da preparação e conectividade entre fontes.
* **Excel:** Uso de funções de texto e lógica para saneamento prévio.

## Conclusão
Dados limpos são a base para narrativas de negócio convincentes.
