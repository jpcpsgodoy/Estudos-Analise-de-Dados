# Módulo 1: Análise de Dados e Business Intelligence

## O Propósito da Análise
Este módulo explora como o analista de dados atua como ponte entre os dados brutos e a tomada de decisão estratégica, utilizando o Power BI para gerar valor ao negócio.

## Conceitos-Chave:
* **Ecossistema de Dados:** Compreensão dos papéis de engenheiros, analistas e cientistas de dados.
* **Ciclo de Vida da Análise:** Do entendimento do propósito (identificação do problema) à implementação das recomendações.
* **Processo Estruturado:** Fases de coleta, processamento (ETL), análise (DAX) e visualização.
* **Interface do Power BI:** Configuração do ambiente Desktop e exploração das ferramentas de navegação.
* **Interatividade:** Criação de visualizações e uso de filtros dinâmicos (*slicers*) para engajamento do usuário.

## Conclusão
A análise de dados eficiente combina competências técnicas com uma cultura orientada a resultados para transformar números em decisões.
