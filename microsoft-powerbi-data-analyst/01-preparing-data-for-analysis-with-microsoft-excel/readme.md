# Curso 01: Preparing Data for Analysis with Microsoft Excel

Este curso aborda a utilizacao do Microsoft Excel em cenarios de negocios para analise de dados, focando na aplicacao de formulas e funcoes essenciais para a preparacao de bases de dados.

O conteudo faz parte da trilha de preparacao para a certificacao Microsoft Power BI Analyst.

## Objetivos do Curso
* Criacao e preparacao de dados no Microsoft Excel para fluxos de analise.
* Utilizacao de formulas e funcoes comuns em planilhas profissionais.
* Preparacao de dados do Excel para integracao e analise no Power BI utilizando funcoes.

## Competencias Desenvolvidas
* **Estruturacao de Dados:** Organizacao de tabelas e intervalos para garantir a integridade da analise.
* **Calculos Dinamicos:** Aplicacao de logica matematica e funcoes de busca para gerar insights.
* **Prontidao para BI:** Tratamento de dados brutos para que sejam facilmente consumidos por ferramentas de Business Intelligence.

## Estrutura de Modulos
* **Modulo 01:** Interface, fundamentos e manipulacao basica de dados.
* **Modulo 02:** Formulas, funcoes essenciais e logica de calculo.
* **Modulo 03:** Organizacao, limpeza e filtragem de dados.
* **Modulo 04:** Funcoes avancadas e preparacao final para Power BI.
