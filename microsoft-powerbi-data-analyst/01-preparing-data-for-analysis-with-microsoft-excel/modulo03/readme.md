# Módulo 3: Preparação e Manipulação de Dados no Excel

Este módulo foca na transformação de dados brutos em bases padronizadas e prontas para análise, utilizando funções de texto, data, hora e lógica.

## Conteúdo Técnico

### 1. Limpeza de Texto
* **Padronização:** Funções `UPPER`, `LOWER` e `PROPER`.
* **Extração:** Uso de `LEFT`, `MID` e `RIGHT` para segmentação de strings.
* **Limpeza:** Função `TRIM` para remoção de espaços redundantes.
* **Agregação:** Função `CONCAT` para unificar dados de múltiplas células.

### 2. Gestão de Data e Hora
* **Lógica Serial:** Compreensão de datas como números de série (sistema 1900).
* **Componentes:** Extração e construção com `DAY`, `MONTH`, `YEAR` e `DATE`.
* **Cálculos Úteis:** `NETWORKDAYS` para prazos em dias úteis e funções dinâmicas `TODAY` e `NOW`.



### 3. Funções Lógicas e Condicionais
* **Tomada de Decisão:** Uso de `IF` (SE) e `IFS` (SES) para automação de resultados.
* **Lógica Composta:** Funções `AND` (E) e `OR` (OU) para múltiplos critérios.
* **Agregações:** `SUMIF`, `AVERAGEIF` e `COUNTIF` para resumos baseados em condições.



## Práticas de Engenharia de Dados
* **Referências Absolutas:** Uso de `$` para fixação de parâmetros em cálculos em massa.
* **Integridade de Dados:** Conversão de fórmulas em valores para otimização de performance e segurança da informação.
* **Tratamento de Exceções:** Inclusão de calendários de feriados em funções de produtividade.
