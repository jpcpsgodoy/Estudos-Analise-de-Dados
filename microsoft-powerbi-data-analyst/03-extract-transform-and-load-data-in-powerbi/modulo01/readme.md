# Módulo 1: Extração de Dados no Power BI (Extract - ETL)

Este módulo foca na primeira etapa do processo ETL (Extract, Transform, Load), abordando a extração e conexão com diferentes fontes de dados utilizando o Power BI Desktop através do estudo de caso da Adventure Works.

## Objetivos de Aprendizado

* Compreender o conceito e importância do processo ETL
* Conectar-se a diferentes tipos de fontes de dados (Excel, CSV, bancos de dados, nuvem)
* Navegar e utilizar a interface do Power BI Desktop
* Importar dados de múltiplas fontes para análise
* Entender os modos de armazenamento (Import, DirectQuery, Dual)
* Avaliar a qualidade dos dados durante a extração

## Principais Conceitos Explorados

* **Processo ETL:** Compreensão das três etapas - Extract (Extrair), Transform (Transformar) e Load (Carregar)
* **Fontes de Dados:** Conexão com arquivos locais (Excel, CSV, XML), bancos de dados (SQL Server, MySQL), serviços em nuvem e APIs
* **Interface do Power BI Desktop:** Navegação nas visualizações Report, Data e Model, além da Faixa de Opções
* **Modos de Conexão:** Diferenças entre Import (dados armazenados localmente), DirectQuery (consulta em tempo real) e Dual (híbrido)
* **Get Data:** Utilização da função principal para conectar e importar dados de diversas fontes

## Habilidades Desenvolvidas

* Identificar e selecionar fontes de dados apropriadas para projetos de análise
* Conectar-se a diferentes tipos de arquivos e bancos de dados
* Importar dados mantendo integridade e estrutura original
* Avaliar preliminarmente a qualidade e estrutura dos dados extraídos
* Escolher o modo de armazenamento adequado baseado nas necessidades do projeto
* Gerenciar múltiplas conexões de dados em um único projeto Power BI
