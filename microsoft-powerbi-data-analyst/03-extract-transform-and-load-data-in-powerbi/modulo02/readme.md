# Módulo 2: Transformação de Dados no Power BI (Transform - ETL)

Este módulo foca na segunda etapa do processo ETL (Extract, Transform, Load), abordando técnicas de transformação e limpeza de dados utilizando o Power Query no Microsoft Power BI através do estudo de caso da Adventure Works.

## Objetivos de Aprendizado

* Compreender a importância da transformação de dados antes da análise
* Navegar e utilizar a interface do Power Query Editor
* Identificar e alterar tipos de dados corretamente
* Remover e renomear colunas para otimizar datasets
* Identificar e resolver erros comuns em dados (valores nulos, duplicatas, tipos inconsistentes)
* Gerenciar transformações através da lista Applied Steps
* Combinar dados usando Append e Merge
* Aplicar diferentes tipos de Joins

## Principais Conceitos Explorados

* **Power Query Interface:** Navegação nas abas Ribbon (Home, Transform, Add Column, View) e compreensão dos painéis de Consultas, Applied Steps e Visualização de Dados
* **Tipos de Dados:** Identificação e conversão entre tipos numéricos, texto, data/hora, booleano e binário para garantir análises precisas
* **Manipulação de Colunas:** Técnicas para remover colunas desnecessárias, renomear com nomes descritivos e promover linhas de cabeçalho
* **Limpeza de Dados:** Métodos para identificar e tratar valores nulos, remover duplicatas e padronizar tipos de dados inconsistentes
* **Applied Steps:** Gerenciamento do histórico de transformações com capacidade de desfazer, reordenar e modificar etapas aplicadas
* **Combinação de Dados:** Uso de Append para empilhar linhas e Merge para integrar colunas de diferentes tabelas
* **Joins:** Aplicação de Left Outer, Right Outer, Full Outer e Inner Joins baseado nas necessidades de análise
* **Join Keys:** Utilização de chaves primárias e estrangeiras para relacionar tabelas

## Habilidades Desenvolvidas

* Importar dados de diferentes fontes (Excel, CSV) para o Power Query
* Aplicar filtros para identificar dados problemáticos
* Substituir valores nulos por valores padrão usando Replace Values
* Remover linhas duplicadas mantendo integridade dos dados
* Converter tipos de dados para garantir consistência
* Utilizar Applied Steps para controle completo do processo de transformação
* Acrescentar queries usando Append para consolidar dados de mesma estrutura
* Mesclar queries usando Merge com diferentes tipos de Joins
* Identificar e utilizar Join Keys para relacionar tabelas
* Preparar datasets limpos, estruturados e consolidados para análise no Power BI
